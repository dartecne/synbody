
Sensor2Midi_0_2

Aplicacion que lee del puerto serie los datos del traje a través del controlador Arduino y módulo BT.
Estos datos se transforman en señales MIDI.
Incluye una interfaz Gráfica para visualizar los datos.

Implementado en y para Processing 1.5.1

Para ejecutar, abrir sensor2midi.pde con Processing (versión 1.5.1)
